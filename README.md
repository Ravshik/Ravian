# Finora

Личный финансовый веб-трекер: счета, операции, бюджеты, цели, долги, регулярные платежи и режим скрытия сумм.

Приложение работает как статический PWA:

- `index.html` — основной интерфейс;
- `manifest.webmanifest` — данные для установки на главный экран;
- `sw.js` — офлайн-кэш;
- `icon.svg` — иконка приложения.

Локальный запуск:

```bash
python -m http.server 4178
```

После запуска открыть:

```text
http://127.0.0.1:4178/index.html
```
