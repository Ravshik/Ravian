# Ravian

Ravian - статическое PWA-приложение для личных путей развития. Оно показывает крупные показатели, прогресс, дневник изменений, фото, достижения и коллекцию побед.

Текущая версия сделана в направлении Apple Style + мягкий Glassmorphism:

- спокойная премиальная основа;
- большие карточки путей;
- крупные числа вместо перегруженных графиков;
- стеклянные поверхности для карточек и навигации;
- локальное хранение данных в `localStorage`;
- полный сброс всех локальных данных;
- добавление фотографий к пути с датой и подписью;
- экспорт и импорт резервной копии JSON.

Файлы:

- `index.html` - основной интерфейс и логика приложения;
- `manifest.webmanifest` - данные для установки PWA;
- `sw.js` - офлайн-кэш;
- `icon.svg` - иконка Ravian.

Локальный запуск:

```bash
python -m http.server 4178
```

Открыть:

```text
http://127.0.0.1:4178/index.html
```
